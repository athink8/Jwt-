# 
### 基于JWT的单点登录实现项目

> 源码丢失，只剩编译后文件，就当作纪念！

> 本项目已实现基本的单点登录实现;
>
> 本系统采用SpringBoot快速搭建；
>
> 即在登录中心登录后，每个子系统都可以同时检测并登录；
>
> 当注册后，其余所有系统都注册；
>
> 主要采用JWT生成token并检验，储存在cookie或redis中；
>
> 使用一堆加密算法，如AES、RSA、MD5等；
>
> 前端采用boostrap搭建，个人认为还挺好看的样子~hhhh
